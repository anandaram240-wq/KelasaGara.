1
Create Supabase Project — Free Database + Auth
Takes 2 minutes. No credit card.
  1. Go to supabase.com → Sign up free with GitHub
  2. Click New Project → Name it kelasagaara → Choose Singapore region (closest to India)
  3. Save your Project URL and anon public key from Settings → API
// Save these — you'll need them
SUPABASE_URL = "https://xxxx.supabase.co"
SUPABASE_ANON_KEY = "eyJhbGciOiJIUzI1NiIs..."
2
Create Database Tables (SQL Editor)
Run this SQL in Supabase → SQL Editor → New Query
-- Workers table
CREATE TABLE workers (
  id          UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  name        TEXT NOT NULL,
  phone       TEXT UNIQUE NOT NULL,
  trade       TEXT,
  district    TEXT,
  taluk       TEXT,
  rate        INTEGER DEFAULT 0,
  exp         INTEGER DEFAULT 0,
  skills      TEXT[],
  verified    BOOLEAN DEFAULT false,
  rating      NUMERIC(2,1) DEFAULT 0,
  reviews     INTEGER DEFAULT 0,
  created_at  TIMESTAMPTZ DEFAULT now(),
  last_seen   TIMESTAMPTZ DEFAULT now()
);

-- Enable Row Level Security
ALTER TABLE workers ENABLE ROW LEVEL SECURITY;

-- Anyone can read workers, only owner can update
CREATE POLICY "Public read" ON workers
  FOR SELECT USING (true);

CREATE POLICY "Auth insert" ON workers
  FOR INSERT WITH CHECK (auth.uid() IS NOT NULL);
3
Enable Phone OTP in Supabase
Supabase has built-in phone auth — enable it in 3 clicks
  1. In Supabase dashboard → Authentication → Providers
  2. Enable Phone provider → Choose Twilio or MessageBird
  3. Free alternative: Use Fast2SMS.com — ₹0 for first 50 SMS/day. Get API key → paste in Supabase SMS settings
  4. Or use MSG91 — Indian SMS gateway, very reliable, free tier available
💡 Supabase built-in OTP means no custom backend — the phone number receives a 6-digit OTP automatically. Your existing OTP UI already works exactly with this flow.
4
Replace localStorage with Supabase in your code
Add these 3 snippets to KelasaGaara HTML
// 1. Add Supabase SDK in <head>
<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>

// 2. Initialise (replace with your values)
const supabase = supabase.createClient(
  'YOUR_SUPABASE_URL',
  'YOUR_ANON_KEY'
);

// 3. Send OTP — replaces your fake OTP send
async function sendRealOTP(phone) {
  const { error } = await supabase.auth.signInWithOtp({
    phone: '+91' + phone
  });
  if (error) showToast('❌ OTP failed: ' + error.message, 'error');
  else showToast('📱 OTP sent to +91' + phone, 'success');
}

// 4. Verify OTP — replaces your verifyOTP()
async function verifyRealOTP(phone, token) {
  const { data, error } = await supabase.auth.verifyOtp({
    phone: '+91' + phone,
    token,
    type: 'sms'
  });
  if (error) { showToast('❌ Wrong OTP', 'error'); return; }
  // User is now authenticated — save to workers table
  await supabase.from('workers').upsert({ phone, ...userData });
}
